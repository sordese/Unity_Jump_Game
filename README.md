# Unity Jump Game

## 프로젝트 소개
Unity를 사용하여 만든 2D 점프 게임입니다.  
플레이어는 장애물을 점프하여 피하며 점수를 획득하는 게임입니다.

---

## 플레이 방법
- **Space** : 점프
- **↓ (방향키 아래)** : 플레이어 위치를 시작 위치로 이동 (즉시 점프 가능)

---

## 주요 기능
- 점프 시스템 (물리 기반)
- 장애물 자동 생성 및 이동
- 장애물 충돌 시 게임 오버
- 장애물 통과 시 점수 증가
- 점수 UI 표시
- 시간이 지날수록 장애물 속도 증가 (난이도 상승)
- 플레이어 위치 즉시 리셋 기능

---

## 사용 기술
- Unity Engine
- C#
- TextMeshPro (UI)

---

## 구현 내용
- Rigidbody2D를 활용한 점프 물리 구현
- 충돌 감지를 통한 게임 오버 처리
- Prefab을 활용한 장애물 자동 생성
- 이벤트 기반 점수 시스템 구현
- 난이도 증가 로직 설계 (속도 증가)
